DAA — Autonomous SRE

Welcome to the official documentation for DAA (v3.0.0). DAA is a modern, open-source Autonomous Site Reliability Engineering platform designed to drastically reduce MTTR (Mean Time To Recovery) by leveraging AI to triage, root-cause, and resolve production microservice incidents automatically.

Note: Version 3.0.0 introduces complete support for Local Git Workspaces and RabbitMQ async task queues.

AI Root Cause Analysis

Ingests stack traces and logs, instantly diagnosing issues with state-of-the-art LLMs.

Auto-Remediation

Generates pull requests or direct fixes for common production exceptions automatically.

High Performance

Built for scale with pluggable queues and databases to fit right into your infrastructure.

Quickstart

Get up and running with DAA in under 60 seconds using Docker. Below is the recommended configuration.

version: '3.8'
services:
  daa:
    image: rutvej1/daa-standalone:latest
    ports:
      - "8000:8080"
    environment:
      - LLM_PROVIDER=google
      - GEMINI_API_KEY=${GEMINI_API_KEY}
      - DAA_DB_PROVIDER=postgres
      - DAA_QUEUE_MODE=rabbitmq
docker run -d --name daa \
  -p 8000:8080 \
  -e LLM_PROVIDER=google \
  -e GEMINI_API_KEY=$GEMINI_API_KEY \
  -e DAA_DB_PROVIDER=none \
  rutvej1/daa-standalone:latest
