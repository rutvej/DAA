# Admin Panel - System Overview

The Admin Panel is a web-based interface that serves as the primary control center for the application. It provides administrators with a comprehensive view of the system's status, including the progress of error log processing, the generated fixes, and the overall health of the services.

## Key Features

-   **Dashboard:** A centralized dashboard that displays key metrics and the status of ongoing tasks.
-   **Log Viewer:** A tool for viewing and searching submitted error logs.
-   **Fix Viewer:** A tool for reviewing the fixes generated by the LLM.
-   **System Health:** A view that monitors the status of the Backend API, Worker Service, and other components.

## Technology Stack

-   **Frontend Framework:** React or Vue.js
-   **Styling:** Tailwind CSS or a similar utility-first CSS framework
-   **API Communication:** RESTful API or GraphQL

## Architecture

The Admin Panel is a single-page application (SPA) that communicates with the **Backend API** to fetch data and perform actions. It is a client-side application that runs in the user's browser and does not have its own backend.
