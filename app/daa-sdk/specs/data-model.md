# Daa SDK Data Model

## 1. Log

The following is the data model for the error log that is sent to the Daa backend API:

-   **message**: The error message.
-   **stack_trace**: The stack trace of the error.
-   **context**: A dictionary of context information.
-   **timestamp**: The timestamp of the error.
-   **token**: The authentication token.
-   **repo_name**: The name of the repository where the error occurred.
