# Infrastructure

This document outlines the services that make up the application infrastructure.

## Services

*   **backend-api**: The main API for the application.

*   **admin-panel**: The front-end for the application.
*   **local-llm**: A local large language model for processing error logs.
*   **database**: A PostgreSQL database for storing application data.
*   **google-gemini**: The Google Gemini LLM for processing error logs.
